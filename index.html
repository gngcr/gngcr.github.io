FaceWork

FaceWork is a Python library for facial alignment, cropping, and morphing.

🔗 Links

  • GitHub Repository
  • PyPI Page

📦 Installation

pip install facework

🧪 Example Use

from facework import FaceCrop

cropper = FaceCrop()
cropper.align_and_crop("input.jpg", "output.jpg")
